# Legend
The first line contains 4 integers (`player.x`, `player.y`, `ghost.x`, `ghost.y`) and a double (`p`).

- `#` = brick wall
- `.` = empty tile
- `I` = fire extinguisher
- `F` = fire
- `E` = exit
- `T` = temporary fire
- `X` = empty cell

- `L` = laser
- `N` = energy cube
- `H` = potion
- `S` = switch
- `D` = door
- `P` = portal

- `h` = hammer
- `w` = fake wall
- `i` = illuminated
